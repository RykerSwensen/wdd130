Block vs Inline

block elements by default take up 100% of the space they have, stretching from right to left as far as they can. In HTML there are already some elements that are block or inline. div, header, main, footer, nav, p, ul, li, section, article, form, and h1-h6

inline elements by default will allow other elements to share the same line as long as there is room, only wrapping to the next line once it's full. These include: a, img, span, button, and label.

The display Property
